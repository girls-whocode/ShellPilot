# Keybindings

```md
| Key | Action |
|-----|--------|
| ↑ / ↓ | Navigate entries |
| → | Open directory / action menu |
| ← | Go up one directory |
| / | Search |
| : | Command action palette |
| ? | Help |
| h | Home |
| t | Trash manager |
| del | Safe delete |
| a | AI explain |
| ^b | Bookmark folder |
| ^j | Next bookmark |
| ^, | Settings |
| q | Quit |
```
