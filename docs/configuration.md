# Configuration

ShellPilot stores configuration in:

~/.config/shellpilot/

markdown
Copy code

### Files

- `config.json` — general settings
- `bookmarks.json` — saved bookmarks
- `state.json` — last directory, layout state
- `models.json` — AI models + metadata

### Config Options

| Setting | Description |
|--------|-------------|
| `ai_enabled` | Enable/disable AI |
| `recursive_search_default` | Default recursive mode |
| `editor` | Override external editor |
| `theme` | (future) |

All files use human-readable JSON.
